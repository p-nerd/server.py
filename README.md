# server.py
Custom web server with python for Learning Purpose

## Dependency
* Python==3.8.10
* There is no pip dependency

## Run
* In Linux/MacOS
    ```
    python3 server.py <port>
    ```
    ```
    python3 server.py 5000 # example
    ```
* In Windows
    ```
    python server.py <port>
    ```
    ```
    python server.py 5000 # example
    ```
